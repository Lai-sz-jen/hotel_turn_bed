# 翻床統計系統 UI 原型

本專案依據「翻床統計軟體需求規格書.pdf」製作飯店房務翻床規劃系統的前端 UI 原型。

## 功能範圍

- 樓層維護迫切度總覽
- 個別房間翻床優先級排序
- 建立翻床計畫
- 建立執行紀錄並依最後房號自動勾選已完成房間
- 報表匯出介面
- Gmail 通知條件與通知對象設定

## 開啟方式

直接用瀏覽器開啟 `index.html` 即可檢視原型，不需要啟動伺服器。

## 檔案說明

- `index.html`：UI 頁面結構
- `styles.css`：版面與視覺樣式
- `script.js`：示範資料與基本互動
- `翻床統計軟體需求規格書.pdf`：需求規格書
